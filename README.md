# Курсова робота — Симплекс-метод

Десктопний застосунок для розв'язання задач лінійного програмування симплекс-методом з підтримкою цілочисельного розв'язання (метод Гоморі).

## Запуск

```bash
npm install
npm start
```

## Стек

- **Electron** - десктопна оболонка
- **Vanilla JS** - вся логіка (без фреймворків)
- HTML + CSS - інтерфейс

## Структура файлів

| Файл | Що робить |
|------|-----------|
| `main.js` | Точка входу Electron, створює вікно |
| `index.html` | HTML-каркас сторінки |
| `simplexSolver.js` | Математика: симплекс + метод Гоморі |
| `render.js` | Відображення таблиць і результатів (View) |
| `controller.js` | Обробка подій, зв'язок даних з UI (Controller) |
| `style.css` | Стилі |
| `bg.js` | Анімований фон (canvas) |

## Як працює

1. Користувач вводить дані (типи вагонів, парк, кількість пасажирів, типи потягів)
2. Контролер збирає дані і будує задачу ЛП: цільову функцію F → max і матрицю обмежень
3. `SimplexModel.solve()` розв'язує неперервну задачу симплекс-методом
4. `SimplexModel.solveInteger()` запускає метод Гоморі для цілочисельного розв'язку
5. View рендерить покрокові симплекс-таблиці в UI

## Метод Гоморі (коротко)

- Якщо симплекс-оптимум має дробові значення - генеруємо "відсічення" (нове обмеження)
- Застосовуємо двоїстий симплекс щоб відновити допустимість
- Повторюємо до отримання цілого розв'язку
